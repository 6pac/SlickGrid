SlickGrid icons ⌂

All SVG icons came from the Material Design Icons project, we only picked the minimum icons required by SlickGrid.
The instruction is to simply import the "slick-icons.css" file and then use any of the CSS classes as shown below by using "sgi sgi-icon-name" (Note: the "sgi" class is also required).

Icon extra utils (available sizes, from 10px to 30px)

Pure CSS icons (credits goes to UnoCSS project, see this article)
